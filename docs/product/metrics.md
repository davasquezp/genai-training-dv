# Metrics (lean)

## North-star (early)
- **Interest registrations per week**

## Funnel (minimum viable analytics)
1) Landing page views
2) Interest page views
3) Submit attempts
4) Submit successes
5) Submit failures (with reason bucket: validation, CORS, network, server)

## Quality metrics
- % submissions with at least 2 styles
- Role distribution (lead/follower)
- Country distribution

## Operational metrics
- API error rate (4xx/5xx) on `/api/dancers`
- CORS/preflight failures (browser-only signal)
